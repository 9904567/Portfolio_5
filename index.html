Educational Philosophy

Empowering Students Through Experiential Learning

My classroom framework is structured on active engagement and exploratory play. Rather than focusing on dry technical jargon, we explore hardware integrations, construct custom Micro:bit arrays, prepare teams for competitive robotics leagues, and utilize responsible AI platforms.

Block-to-Text Path

Providing a logical stepping-stone from drag-and-drop block interfaces into standard textual coding syntax (Python, JS).

Physical Circuitry

Connecting theory to physical computing using IoT modules, breadboards, Arduino microcontrollers, and servo assemblies.

AI & ML Literacy

Teaching prompt engineering, machine learning foundations, and moral responsibilities using Teachable Machine and AI platforms.

Competitive Coaching

Guiding teamwork, iterative mechanical improvements, and standard programming documentation for VEX robotic tournaments.

Grades K - 5

Creative Block Coding & Robotics Play

Hardware: LEGO Spike Prime & Scratch 3.0

Key Educational Pillars

  • Computational logic using visual drag-and-drop puzzles
  • Understanding sequential loops and conditional variables
  • Constructing physical LEGO models with smart sensor elements

Sample Project Outcome

The Smart Recycling Bot

Students assemble a physical waste sorting vehicle and code color sensors to distinguish recyclables from organic waste, receiving a visual success badge.

Grades 6 - 8

Microcontroller Prototyping & Dynamic Web

Hardware: BBC Micro:bit & Python Basics

Key Educational Pillars

  • Transitioning from Scratch block stacks into raw Python scripts
  • Wiring breadboards, active buzzers, and LED display matricies
  • Basic HTML/CSS styling for student-driven showcase sites

Sample Project Outcome

IoT Digital Greenhouse Monitor

Students program a BBC Micro:bit circuit to monitor real moisture values, illuminating a warn lamp and sounding active alarms if irrigation limits fail.

Grades 9 - 12

Autonomous Systems, VEX Athletics & ML

Hardware: Arduino (C++), VEX V5 Robotics & AI Models

Key Educational Pillars

  • Real procedural programming syntax using Arduino C++ modules
  • Training personalized ML classifiers utilizing webcam feeds
  • Algorithmic debugging, error logs, and structural design journals

Sample Project Outcome

Gesture-Controlled Smart Rover

Students train a personal ML model to identify specific hand positions, triggering immediate wireless steering packets to an autonomous rover vehicle.
